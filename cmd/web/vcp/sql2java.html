SQL 建表语句转 Java 实体类

粘贴一段 CREATE TABLE 语句，自动生成对应的 Java 实体类，把字段名转成驼峰属性并匹配 Java 类型。解析在浏览器本地完成，建表语句不会上传。

复制

它做什么

读取 MySQL、Oracle 等数据库的 CREATE TABLE 语句，按每个字段生成同名的 Java 属性：下划线列名转成小驼峰（如 address_id → addressId），并把 SQL 类型映射成 int、String 等 Java 类型，省去手写实体类的重复劳动。点“示例”可载入一段建表语句看效果。

生成结果是起点而非终稿：复杂类型（如 decimal、datetime、tinyint(1)）的映射、注解、getter/setter 仍需你按项目规范确认。
