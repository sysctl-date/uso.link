Excel / CSV 转 JSON

把从 Excel 或 CSV 复制来的表格数据，按行转成 JSON 对象或数组，转换在浏览器本地完成、不上传。

  • 按行转成对象
  • 按行转成数组
复制结果

怎么用

从 Excel 或 CSV 里复制带表头的数据粘进输入框，点“转换成Json”即可。默认按制表符分隔（直接从 Excel 复制就是这种格式），如果是逗号或其他分隔符的 CSV，在分隔符框里填上对应字符。不确定格式时可先点“示例Demo”看一组样例。

第一行会被当作字段名。通过下拉菜单可选“按行转成对象”（每行生成一个键值对象）或“按行转成数组”（每行生成一个值数组）。整个转换在本机完成，表格数据不会上传。
