网址桌面快捷方式生成

填入名称和网址，生成一个 Windows 桌面快捷方式文件并下载，双击就能在浏览器里打开指定网址。

名称:
网址

怎么用

在“名称”里写快捷方式显示的标题，在“网址”里填要打开的链接，点“创建快捷方式并下载”，会下载一个 Windows 快捷方式文件（.url）。把它放到桌面，双击就会用默认浏览器打开这个网址。

生成的是一个指向网址的链接文件，不抓取也不保存目标页面内容；这种快捷方式主要适用于 Windows，其他系统打开网址的方式不同。
