SQL 代码格式化

把挤成一行或缩进混乱的 SQL 语句按关键字重新缩进换行，看清查询结构。语句在浏览器本地处理，不会上传。

怎么用

粘进一段写在一行或缩进杂乱的 SQL，点“SQL代码格式化”，工具会按关键字断行并对齐缩进，让 SELECT、FROM、WHERE、JOIN 等层次清楚，方便核对长查询。

它只重排版面，不解析或执行语句，也不连接数据库；适用于 MySQL、Oracle、SQL Server 等常见数据库的语法。如果只想把 SQL 压成一行，用站内的 SQL 压缩工具更合适。
