C++ 代码格式化

把杂乱、压缩的 C++ 代码按统一缩进重新排版并语法高亮，方便阅读和检查，排版在浏览器本地完成。

复制

怎么用

把压缩成一行、或缩进混乱的 C++ 代码粘进输入框，点“C++代码格式化”，工具会按统一规则重新缩进、对齐大括号，并给关键字、字符串语法高亮，结果更易读。整理别人的代码或还原压缩过的源码时很方便。

它只调整排版和外观，不会编译或检查语法对错，也不会改变程序逻辑。整理完仍需自行确认代码能正常编译运行。
