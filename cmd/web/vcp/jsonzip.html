JSON 压缩、转义与格式化

把 JSON 压缩成单行、添加或去除转义、或重新缩进展开，全部在浏览器本地完成，数据不会上传。

复制

三种操作怎么选

压缩把 JSON 去掉所有空格和换行变成单行，减小体积、便于在请求体或配置里传输；格式化反过来，按层级缩进展开，方便阅读和排查结构。

转义给双引号、反斜杠等字符前加上 \，让一段 JSON 能作为字符串安全地嵌进另一段 JSON 或代码里；去除转义则还原成原始 JSON。

注意顺序：被转义过的内容直接格式化会失败，需要先“去除转义”拿回真正的 JSON，再做格式化。
