XPath 在线解析与测试

粘贴一段 HTML，用 XPath 表达式定位并提取其中的元素，也能一键抽取页面里的全部图片或链接。解析在浏览器本地运行。

复制结果

怎么用

把目标网页的 HTML 源码贴进输入框，在表达式栏写 XPath（如 //div[@class='item']/a/@href）后点「xpath匹配」，即可看到命中的节点。写爬虫或采集规则前，用它先验证表达式能不能选中想要的内容，比反复跑脚本快得多。

「解析图片」「解析链接」是两个快捷方式，分别抽取 HTML 里所有 <img> 的地址和 <a> 的链接；不确定怎么写时可点「示例数据」载入一段样例对照。
