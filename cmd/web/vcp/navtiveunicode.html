这两种写法的区别

Native 指原始可读的文本，比如直接写出的中文“你好”。Unicode 形式则把每个字符换成它的码点数字，写成 &#20320;&#22909; 这样的 HTML 数字实体。两者表示的是同一段内容，只是写法不同。

当文件或传输通道只接受 ASCII、或想避免中文在不同编码间出现乱码时，可以先转成 Unicode 实体；需要阅读时再转回 Native。还原时请保持 &#数字; 的完整格式，否则无法正确解析。
