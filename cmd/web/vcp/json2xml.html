XML 与 JSON 互转

把 XML 转成 JSON，或把 JSON 转成 XML，并可选格式化输出。转换在浏览器本地完成。

在两种数据格式间转换

对接老系统时经常遇到 XML，而新接口多用 JSON，本工具帮你在两者之间来回转。点“Xml转Json”把节点转成键值对象，点“Json转Xml”反向生成；勾选“格式化Json”可让输出带缩进便于阅读。不确定输入格式时，可点“Xml示例”或“Json示例”看一眼。

两种格式的表达能力并不完全对应：XML 的属性、命名空间、混合文本，以及 JSON 的数组、布尔/数值类型，在转换时会按固定规则映射，复杂文档转完后建议核对一遍是否符合预期。
