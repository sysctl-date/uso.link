怎么用这些地址

上表列出各家公共 DNS 的主、备地址。把其中一组填进操作系统或路由器的 DNS 设置里（主、备各填一个），就能换掉运营商默认的解析服务器。国内访问通常 114、阿里、DNSPod 延迟更低；Google（8.8.8.8）、Cloudflare（1.1.1.1）、OpenDNS 等海外地址在国内不一定稳定。

更换 DNS 能改善的是“域名解析”这一步，可能解决某些网站打不开、被运营商插入广告或遭到 DNS 劫持的问题，但它不会提升你的实际带宽。表中地址会随各家服务调整，使用前以官网链接为准。
