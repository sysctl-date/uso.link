在 JSON 和 URL 参数之间切换

调试接口时常要在两种写法间来回倒：把 {"name":"tom","age":18} 这样的 JSON 拼成 name=tom&age=18 贴到地址栏，或者把抓到的查询串还原回 JSON 看清结构。本工具做的就是这件事。

它做的是简单的键值拆分，适合平铺的一层参数；嵌套对象、数组这类复杂结构 GET 参数本身也表达不了，结果需自行核对。注意它不做 URL 编码，参数值里若含中文或 &、= 等特殊字符，仍需另行转义。
