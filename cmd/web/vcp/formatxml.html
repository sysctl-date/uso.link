XML 格式化与压缩

把挤成一行的 XML 重新缩进换行看清层级，或把 XML 压成一行去掉多余空白。处理在浏览器本地完成，内容不会上传。

格式化和压缩的区别

“xml格式化”按标签层级重新加上换行和缩进，让嵌套结构一目了然，方便阅读和排查；“xml压缩”去掉标签之间的换行和空白，把内容压成一行，减小体积、便于传输或存储。两者只改排版，不改变标签和数据本身。

需要注意的是，元素文本内容里的空白在某些场景下是有意义的，压缩可能改变这部分空白；对空白敏感的数据先确认再压缩比较稳妥。
