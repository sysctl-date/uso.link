把 HTML 拼进 C# 或 JSP

在 C#（.NET）或 JSP 里逐行输出 HTML 时，需要把每行包成字符串、转义内部引号、再用输出语句拼接。这个工具把粘贴进来的 HTML 自动转成对应语言的字符串拼接代码，省去手工加引号和转义的麻烦。

转换是单向的：从 HTML 生成代码，不做反向还原。点「Html转为C#代码」或「Html转为JSP代码」得到结果，复制后按项目风格调整即可。
