把 HTML 拼进 PHP

在 PHP 里用 echo 输出 HTML 时，整段标记要包成字符串、转义内部的引号，行数一多手工处理很费劲。这个工具把粘贴进来的 HTML 自动转成 PHP 的字符串拼接代码，省去逐行加引号和转义。

转换是单向的：从 HTML 生成 PHP 代码，不做反向还原。点「Html转PHP代码」得到结果，复制到源文件后按需调整即可。
