JSON 生成 C# 实体类

把一段 JSON 解析成对应的 C# Model 类，自动推断字段类型和嵌套结构。生成在浏览器本地完成。

复制C#实体类校验Json

从接口返回快速建模

对接 API 时，常常需要照着返回的 JSON 手写一遍 C# Model 类。本工具替你做这件事：按 JSON 的字段名和值类型生成对应的属性，嵌套对象会拆成独立的类，数组会推断出元素类型。

JSON 必须语法合法才能生成，遇到错误会提示而不会硬转。生成结果是起点而非终点——字段名大小写、可空类型、日期格式等仍需按你的项目规范手动调整。
