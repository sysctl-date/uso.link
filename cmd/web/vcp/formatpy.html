Python 代码格式化

把缩进混乱的 Python 代码按统一缩进重新排版，便于阅读和检查。代码在浏览器本地处理，不会上传。

怎么用

把缩进杂乱的 Python 代码粘进输入框，选好缩进方式和换行规则，点“Python代码格式化”就会重新排版输出。它只调整缩进和换行，不改动代码逻辑。

需要特别提醒：Python 用缩进来划分代码块，缩进改变可能影响语义。这个工具按通用规则重排版面，结果建议自己核对一遍；要严格遵循 PEP 8，本地用 black、autopep8 这类专门工具更可靠。
