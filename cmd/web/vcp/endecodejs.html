JavaScript eval 加密与解密

把 JS 代码打包成 eval 形式让其不易阅读，或把这类代码还原成可读源码，全部在浏览器本地完成。

复制

它做的是混淆，不是加密

这里的“加密”用的是 eval 打包：把源码压缩、改写成一段由 eval 执行的字符串，看起来杂乱、不便直接阅读。它能挡住随手看代码的人，但不提供真正的保密——代码在浏览器里仍要被解开执行，因此总能被还原。点“JS解密”就能把这类代码恢复成可读形式。

常见用途是给前端脚本做轻度保护、缩小体积或快速看懂别人压缩过的代码。涉及密钥、口令等真正需要保密的内容，不能依赖它。
