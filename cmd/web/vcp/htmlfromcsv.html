Excel / CSV 转 HTML 表格

把从 Excel 复制的内容或 CSV 文本，按指定分隔符转成可直接用的 HTML 表格代码，转换在浏览器本地完成，内容不会上传。

复制

把表格数据变成网页表格

从 Excel 里复制的单元格默认是用 Tab 分隔的文本，导出的 CSV 则用逗号分隔。把这类文本粘进来、选对分隔符，工具就按行列拆开并生成对应的 <table> 代码，省去手工敲标签。

注意 CSV 的引号规则：被双引号包住的字段里，逗号属于内容而不是分隔符，字段内的双引号用两个 "" 表示——上方示例数据已经覆盖了这几种情况。分隔符选错会导致拆列错位，遇到特殊分隔符可在「自定义」里填写。
