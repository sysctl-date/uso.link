PHP 代码格式化

把缩进混乱或压成一行的 PHP 代码按统一缩进重新排版，并对语法着色，便于阅读和检查。代码在浏览器本地处理，不会上传。

怎么用

把缩进错乱或挤成一行的 PHP 代码粘进输入框，点“PHP代码格式化”，工具会重新缩进排版并给语法着色，方便快速看清结构、定位明显的书写问题。它只重排版面，不执行代码，也不改动逻辑。
