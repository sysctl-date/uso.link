JSON 与 YAML 互转

把 JSON 转成 YAML，或校验 YAML 并转回 JSON。转换在浏览器本地完成。

YAML 和 JSON 是什么关系

两者都用来表达对象、数组、字符串这类数据，能等价互转。区别在写法：JSON 靠大括号、方括号和引号划分结构，适合程序传输；YAML 用缩进和换行表达层级，省掉了大量括号和引号，读写更省眼，所以 Kubernetes、CI 配置等常用 YAML。

使用时注意

YAML 对缩进敏感，且只能用空格、不能用 Tab，层级错乱就会解析失败——点“YAML格式校验/转JSON”可先验证再转。把现成的 JSON 配置点“JSON转YAML”即可得到更易读的版本。
