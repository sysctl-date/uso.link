收录各高校与中国教育网（CERNET）的 DNS 服务器地址，方便校园网用户对照选用。

关于这份清单

下表是各高校及教育网节点对外提供的 DNS 服务器地址和归属单位。校园网内访问教育网资源时，使用就近的教育网 DNS 通常解析更快。

这些地址多由各高校网络中心维护，主要面向教育网内部，未必对校外网络开放，也可能随时调整。校外用户建议改用 114、阿里等公共 DNS。
