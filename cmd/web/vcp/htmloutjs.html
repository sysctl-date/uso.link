HTML 和 JS 之间互相还原

用 JavaScript 动态生成页面时，HTML 需要写成带引号、做好转义的字符串才能用 document.write 或 innerHTML 输出。这个工具双向处理：「Html转成Js」把标记拼成 JS 字符串，「Js转成Html」把这类拼接代码反向还原成可读的 HTML 页面。

反向还原对调试别人写的、被拼成一长串字符串的 JS 输出尤其有用，能快速看清它最终生成的页面结构。
