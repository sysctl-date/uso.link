HTTP 状态码在线查询

输入网址，由服务器代为发起请求，返回该页面的 HTTP 状态码、服务器 IP 和响应头，用来判断页面是正常、跳转还是出错。

常见状态码怎么读

查询由本站服务器对目标网址发起，因此即使该网址在你本地打不开，只要服务器能访问，也能拿到结果。返回的状态码反映服务端对这次请求的处理结果：

200请求成功，正常返回网页
301永久重定向，资源已换到新地址
302临时重定向，资源暂时在别处
404请求的页面不存在
500服务器内部错误
503服务不可用，常见于过载或维护
