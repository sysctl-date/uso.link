JSON 在线解析（左右视图）

左边编辑 JSON，右边实时渲染成可折叠的高亮视图。解析在浏览器本地完成，数据不会上传到服务器。

边改边看

左侧文本框输入 JSON，右侧会同步渲染成带语法高亮、可逐层折叠的树形视图，适合快速看清一段陌生 JSON 的结构。改完点“压缩/格式化”在两种排版间切换，点“转Xml/Json”做格式互转。

解析失败时会给出具体的错误位置，常见原因是键名漏了双引号、末尾多了逗号，或字符串里混进了 Tab 制表符。
