HTML 在线富文本编辑器

在浏览器里所见即所得地排版文章，随时切换查看对应的 HTML 源码，适合写网页内容、公众号或论坛帖子。

怎么用

直接在编辑区里输入和排版，用上方工具栏调整字体、颜色、对齐、列表和表格。点工具栏的“源码”按钮可以看到并直接编辑生成的 HTML，把它复制到自己的网页或后台即可。

编辑过程都在本机浏览器进行，内容不会上传。本编辑器关闭了图片、视频、Flash 上传，插入图片请改用图床外链。
