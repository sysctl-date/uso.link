HTML 转 ASP / Perl / SWS / VB.NET 代码

把一段 HTML 拼接成 ASP、Perl、SWS 或 VB.NET 里直接输出的字符串代码，转换在浏览器本地完成，内容不会上传。

复制

把 HTML 嵌进后端代码

在 ASP、Perl、VB.NET 这类后端代码里逐行输出 HTML 时，要手动给每一行加引号、转义内部的引号、再用语言的输出语句拼起来，行数一多很容易出错。这个工具按目标语言的语法，把粘贴进来的 HTML 自动转成对应的字符串拼接代码。

转换是单向的：从 HTML 生成代码，不做反向还原。选好语言点对应按钮即可，生成结果可直接复制到源文件中再按需调整缩进。
