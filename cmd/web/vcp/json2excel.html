把 JSON 数据摊成表格

最适合处理的是“对象数组”——每个对象是一行，对象的键作为列头。本工具把这样的 JSON 拉平成表格，默认用制表符分隔（可直接粘进 Excel），也可以在分隔符框里改成逗号等导出标准 CSV，再点“下载Csv文件”保存。

结构不规则、深层嵌套的 JSON 不一定能干净地变成二维表，转换前可点“示例Demo”看一眼期望的数据格式。
