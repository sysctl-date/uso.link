关于 UBB 代码

UBB（也写作 BBCode）是论坛里用方括号写的标记，例如 [b]粗体[/b]、[url=...]链接[/url]。它本质上是受限的 HTML 替代品：论坛允许用户用 UBB 排版，但禁止直接发 HTML，以避免脚本注入等风险。

这个工具双向转换：「Html转UBB」把网页标记转成论坛能识别的 UBB，「UBB转Html」把帖子里的 UBB 还原成 HTML 预览。只覆盖加粗、斜体、链接、图片、列表等常见标签，UBB 没有对应写法的复杂 HTML 不保证完整保留。
