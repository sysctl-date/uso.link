JSON 格式化与压缩

格式化、压缩、转义 JSON，并校验语法是否合法。处理在浏览器本地完成，数据不会上传到服务器。

请输入要压缩的Json字符串

这个工具能做什么

JSON 是前后端和各种程序之间交换数据的常用文本格式。压缩后的 JSON 挤成一行不便阅读，本工具把它按层级缩进展开；反过来也能去掉空白压成一行以减小体积。校验会指出语法错在哪，便于排查多逗号、键名漏引号这类问题。

校验不通过的常见原因

键名和字符串值必须用双引号、最后一项后面不能多逗号、注释不属于标准 JSON。如果是从代码里复制的字符串，记得先用“去除转义”把多余的反斜杠还原，再做格式化。
