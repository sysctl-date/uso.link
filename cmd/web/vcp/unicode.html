关于 Unicode 转换

Unicode 给每个字符分配了一个码点，转换只是把同一个字符在不同写法之间切换，并不是加密——任何人都能直接还原。常见场景是把中文转成 中文 这类转义形式，方便写进 JavaScript、JSON 字符串或源码里避免乱码；也可以反过来把转义还原成可读的中文。

&#XXXX 是 HTML 里用十进制码点表示字符的写法，适合直接放进网页；而 \uXXXX 是十六进制，多用于程序代码。两者对应的是同一码点，按使用环境选择即可。
