Escape / Unescape 在线编码解码

用 JavaScript 的 escape() 把字符串转成 %xx 形式，或用 unescape() 还原，常用于解决 URL、脚本里中文乱码，转换在浏览器本地完成。

关于 Escape 编码

escape() 是一种编码而非加密——它只是把字符替换成 %xx 这样的可打印形式以便安全传输，谁都能用 unescape() 还原，不提供任何保密性。它常用来让中文、特殊符号顺利穿过 URL 或 JS 字符串，避免乱码。

编码时，ASCII 字母、数字以及 * @ - _ + . / 保持不变，其余字符被替换成转义序列（非 ASCII 字符用 %uXXXX）。注意 escape 是早期遗留方法，对 URL 处理已不推荐，新代码更适合用 encodeURIComponent()；本工具主要用于和旧系统对接或快速排查乱码。
