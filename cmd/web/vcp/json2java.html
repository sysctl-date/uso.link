从 JSON 生成 JavaBean

填好类名和包名，贴入 JSON，工具会按字段名和值类型生成对应的 JavaBean，嵌套对象拆成独立的类，可直接打包成源码下载。JSON 语法不合法时会提示而不会硬转。

字段撞上关键字怎么办

如果 JSON 的字段名正好是 Java 关键字（如 default），又不能改名，可以改用合法的字段名再用 Gson 的 @SerializedName 注解绑定原始键。例如把：

private List<Default> default;

改写成：

@SerializedName("default")

private List<Default> mdefault;
