HTML 与 Markdown 互转

把 Markdown 渲染成 HTML，或把 HTML 反向转成 Markdown，转换在浏览器本地完成，内容不会上传。

复制

在两种标记之间切换

Markdown 用 #、*、- 这类轻量符号写排版，可读性好，但发布到网页或邮件时通常需要 HTML。这个工具双向转换：「Markdown转Html」把文档渲染成网页代码，「Html转Markdown」把现成的 HTML 反推回简洁的 Markdown，方便迁移到博客或 README。

不确定语法时点「Markdown示例」可载入一段示范。反向转换时，HTML 里没有 Markdown 对应写法的结构（如复杂表格、内联样式）可能被简化或保留为原始标签。
