图片与 Base64 互转

把图片编码成 Base64 Data URI，或把 Base64 字符串还原成图片。读取与编码在浏览器本地完成，图片不会上传到服务器。

1、图片转换成Base64编码，点击“选择图片...”按钮，上传要转换的图片,即可自动生成Base64编码（如果图片尺寸超过2M,请耐心等待3-5秒钟左右）

2、Base64编码转换成图片，请在下面输入框,粘贴base64编码,然后点击“Base64转图片”按钮，即可转为图片

把图片内嵌进代码

Base64 把图片字节编码成一段文本（Data URI），可以直接写进 HTML、CSS 或 JSON，省掉一次单独的图片请求。它适合体积小的图标、背景这类资源；编码后体积会比原图大约三分之一，大图反而拖慢页面，不建议这样用。支持 PNG、GIF、JPG、BMP、ICO 等常见格式。

怎么用

在 CSS 里写 background-image: url("data:image/png;base64,iVBOR...")，在 HTML 里写 <img src="data:image/png;base64,iVBOR...">。反过来，把一段以 data:image/...;base64, 开头的字符串粘进文本框，点“Base64转图片”即可还原预览。
