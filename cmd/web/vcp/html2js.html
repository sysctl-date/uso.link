HTML 与 JS 互转

把 HTML 拼接成 JavaScript 字符串或数组，也能把这类 JS 代码还原成 HTML，转换在浏览器本地完成，内容不会上传。

复制

HTML 和 JS 字符串之间换来换去

用 JavaScript 动态写入 HTML 时，要把整段标记转成带引号、做好转义的字符串才能赋给 innerHTML 或用 document.write 输出。这个工具帮你两个方向都做：把 HTML 拼成 JS 字符串（或按行拆成 JS 数组），也能把这类 JS 代码反向还原回可读的 HTML。

「Html转为JS」生成字符串拼接，「Html转为JS数组格式」按行输出数组便于后续 join，「JS转为Html」做反向还原。粘贴内容点对应按钮即可。
