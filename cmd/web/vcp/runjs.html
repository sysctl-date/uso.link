HTML / CSS / JS 在线运行

把一段 HTML（可含内联 CSS 和 JavaScript）粘进来，点一下就在新窗口里运行并预览效果，方便快速试代码。运行在你自己的浏览器里。

复制

怎么用

把完整的 HTML 片段粘进文本框，点“调试预览”，工具会新开一个窗口，把你的代码当成一个网页运行出来，内联的 CSS 样式和 <script> 里的 JavaScript 都会生效，适合临时验证一段布局或脚本是否正常。

代码在新窗口里执行，所见即所得。如果脚本依赖 jQuery 等外部库，记得在代码里自己用 <script src="..."> 把它引进来。
