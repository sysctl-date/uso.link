URL 网址 16 进制编码

把网址里的字符转成 % 加十六进制的百分号编码形式，编码后的地址仍可直接粘到浏览器地址栏访问。转换在浏览器本地完成。

复制

这是编码，不是加密

它把网址逐字符转成 % 加十六进制的百分号编码（即 URL 编码 / percent-encoding），例如把字母、符号都换成 %xx 的形式。这只是编码，浏览器会自动还原，任何人都能解码，并不能保密，只是让网址看起来不直观。

常见用途是混淆一下展示出来的链接，或确保特殊字符在 URL 中安全传输。编码后的地址直接粘到地址栏仍能正常打开，因为浏览器在请求前会把它解码回原网址。
