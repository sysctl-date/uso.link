JSON 生成 Go 结构体

把一段 JSON 解析成对应的 Go struct，自动推断字段类型并带上 json 标签。生成在浏览器本地完成。

复制Go结构体 校验Json

照着 JSON 写 struct

在 Go 里反序列化 JSON 需要先定义匹配的 struct。本工具按 JSON 的字段名和值类型生成 struct，字段名转成导出的大驼峰，并自动加上 json:"原字段名" 标签，保证能正确解码。

勾选“内联类型定义”时，嵌套对象会直接写成匿名嵌套结构；不勾则拆成多个具名 struct，复用时更清晰。类型按样例值推断，遇到 null 或可能为空的字段，生成后建议自己确认是否要改成指针或 omitempty。
